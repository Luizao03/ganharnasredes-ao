# GanharNasRedes-AO (base HTML/CSS/JS)

Estrutura base do projeto feita para HTML, CSS e JavaScript puro com Firebase.

Instruções rápidas:
1. Cria um repositório no GitHub `Luizao03/ganharnasredes-ao` e cola estes ficheiros.
2. Configura o Firebase (Auth Phone, Firestore, Storage, Functions) e coloca as credenciais em `js/firebase-config.js`.
3. Serve com Firebase Hosting ou GitHub Pages.
